# Zabbix Media Type — MAX

Отправка уведомлений Zabbix в мессенджер [MAX](https://max.ru) через Webhook.

---

## Требования

- Zabbix 7.0+
- Бот на платформе MAX

---

## Получение токена бота

1. Откройте [business.max.ru](https://business.max.ru/self)
2. Перейдите: **Чат-боты → Интеграция → Получить токен**
3. Включите **«Добавление в групповые чаты»**, если нужна отправка в чат

---

## Установка

### 1. Импорт медиатипа

`Administration → Media types → Import` — загрузите файл `zbx_export_mediatypes.yaml`

### 2. Настройка медиатипа

Откройте импортированный медиатип **MAX** и заполните параметр:

| Параметр | Значение | Обязательный |
|----------|----------|:---:|
| `Token` | Токен бота MAX | ✅ |
| `UserId` | ID пользователя MAX | ⚠️ |
| `ChatId` | ID группового чата MAX | ⚠️ |
| `Format` | `html` или `markdown` | — |

> ⚠️ Укажите **только один** из параметров: `UserId` **или** `ChatId`. Указание обоих вызовет ошибку.

### 3. Настройка пользователя

`Administration → Users → <пользователь> → Media → Add`

- **Type**: `MAX`
- **Send to**: ID пользователя или чата

#### Как узнать ID чата

Добавьте бота в чат и выполните запрос:

```bash
curl -X GET "https://platform-api.max.ru/chats" \
  -H "Authorization: <токен>"
```

В ответе найдите нужный чат и скопируйте значение `chat_id`.

---

## Форматирование сообщений

Параметр `Format` поддерживает два режима:

**HTML**: `<b>жирный</b>`, `<i>курсив</i>`, `<code>код</code>`, `<a href="...">ссылка</a>`

**Markdown**: `**жирный**`, `*курсив*`, `` `код` ``, `[ссылка](url)`
